# r-happiness-paradox-project
Analyse du "paradoxe du bonheur" en France via stats descriptives en R (données ESS). Le revenu n'explique pas la satisfaction de vie (ρ=0.30, p=0.69 en régression) : santé, aisance ressentie et liens sociaux dominent. Inclut code R, graphiques et indicateur objectif Mean410. Projet ENSAE.
